# ✅ To-Do Listify

A task management web application designed to organize daily tasks efficiently with a clean modern UI.

---

## 🚀 Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark completed
- Filter tasks
- Sort tasks
- Progress tracking
- Dark / Light mode
- Thai / English language support
- Data saved with localStorage

---

## 💻 Tech Stack

- React
- JavaScript
- CSS3
